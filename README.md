Browse two websites side-by-side in a single tab. Ideal for research, coding, or watching videos while you work.
- **Toggle**: Click the Split icon in the navbar.

## üöÄ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS)
- [Ollama](https://ollama.com/) (For AI features)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ibrahimsamad/isb-browser.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```
